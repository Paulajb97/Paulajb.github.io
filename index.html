DataLoader
Unity - C# + C++ + Bases de datos
DataLoader es una herramienta para Unity, y para un futuro Unreal, que guarda configuraciones tanto en una base de datos local como en un xml, a elección del usuario. Estas configuraciones se pueden establecer en cualquier actor que tenga el recipiente adecuado. En este caso, el recipiente es una variable del tipo DatabaseRow.
